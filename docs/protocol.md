# 通信协议草案

本文档记录电源控制板与上位机之间的通信协议草案，后续根据实际硬件和固件实现更新。

## 数据帧建议

```text
帧头 | 长度 | 命令字 | 数据载荷 | 校验 | 帧尾
```

| 字段 | 说明 |
| :--- | :--- |
| 帧头 | 固定标识，用于快速同步数据帧。 |
| 长度 | 命令字和数据载荷的总长度。 |
| 命令字 | 表示控制、查询、配置或告警类型。 |
| 数据载荷 | 不同命令对应的参数。 |
| 校验 | CRC8 / CRC16 / 累加和，按可靠性要求选择。 |
| 帧尾 | 可选字段，用于辅助调试和帧同步。 |

## 命令示例

| 命令 | 方向 | 说明 |
| :--- | :--- | :--- |
| `0x01` | 上位机 -> 控制板 | 设置某一路电源开关。 |
| `0x02` | 上位机 -> 控制板 | 查询全部通道状态。 |
| `0x03` | 控制板 -> 上位机 | 返回电压、电流、温度等状态。 |
| `0x04` | 控制板 -> 上位机 | 上报告警或保护触发。 |
| `0x05` | 上位机 -> 控制板 | 清除可恢复故障。 |

## 状态量建议

- 每路输出开关状态。
- 输入电压。
- 每路输出电压。
- 每路输出电流。
- 板载温度。
- 故障码。
- 固件版本。
